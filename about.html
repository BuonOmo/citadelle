À propos de l'algorithme simulant les batailles

Généralités

L'algorithme imaginé ici essaye de reproduire le plus fidèlement possible une bataille de Kingdom. Il prend par exemple en compte l'évolution du nombre de duels simultanés au cours du temps etc.

Le code javascript de ce script est accessible à tous grâce aux outils de développement de votre navigateur (fichier fonctions.js). Si le javascript vous est étranger mais que vous avez une question portant sur l'interprétation du simulateur (par exemple : "le script sélectionne-t-il à chaque fois les deux duelistes aléatoirement ?"), n'hésitez pas à me contacter.

Merci à Simoons pour son énorme contribution à l'amélioration des performances de l'algorithme.

Formules utilisées

Le principal défi à relever pour obtenir un script véritablement fidèle au jeu est de trouver des formules justes dans la mesure où beaucoup de règles ne sont pas officielles et stagnent au stade d'hypothèses.

Les formules utilisées par ce script pour déterminer le nombre de duels ainsi que les dégâts infligés à chaque attaque proviennent de la page du groupe IPK (Infos Pour Kingdom).
